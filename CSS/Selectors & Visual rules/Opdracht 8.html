Met Cascade wordt aangegeven dat er trapsgewijs gewerkt wordt, dit komt er op neer dat bij de gelijke prioriteit (specificity) de laatste input de andere styles zal overrulen. De hoogste specifity gaat altijd voor.

je hebt daarin 4 klasses gerangshikt van hoog naar laag zijn deze:

  1. inline selectors
  2. id selectors
  3. class selectors& attribute selectors
  4. element selectors
In de "opdracht.css" file kun je zien dat er 2 selectors proberen de kleur te beinvloeden, maar omdat de specificity gelijk is (het zijn beide element selectors) zal de laatste input (rood) de anderen overrulen.

Met inheritence wordt aangegeven dat child elementen dezelfde style selectors hebben als hun parent element.Sommige elements zoals A voor links hebben standaard inheritence. bijvoorbeeld deze link naar google.nl zal standaard blauw zijn.

Naast de ingebouwde inheritence van bepaalde elementen kun je ook zelf inheritence properties aanmaken voor bepaalde attributen van elementen. dat kan met de volgende commands:

  • inherit: Sets the property value applied to a selected element to be the same as that of its parent element. Effectively, this "turns on inheritance"
  • initial: Sets the property value applied to a selected element to the initial value(default value)of that property.
  • unset: Resets the property to its natural value, which means that if the property is naturally inherited it acts like inherit, otherwise it acts like initial.
