Opdrachten

  1. The internet is millions of computers connect by thousands of networks while the world wide web is billions of pages that can be accessed through the internet. A collection of related web pages.
  2. HTML lets you create the structure of a website. CSS lets you make the website look nice. JavaScript lets you change HTML and CSS
  3. An HTML element is defined by a start tag, some content, and an end tag:
    • The element is a container for metadata (data about data) and is placed between the tag. HTML metadata is data about the HTML document. Metadata is not displayed.
    • The tag represents the root of an HTML document. The tag is the container for all other HTML elements (except for the tag).
    • The tag defines the document's body. The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. Note: There can only be one element in an HTML document.
